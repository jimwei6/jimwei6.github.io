This is a class project for CPSC310 (Introduction to Software Engineering) at The University of British Columbia. The project is composed of three parts:

  • parsing specifically structured HTML and JSON files into query-able data structures
  • handling requests and filtering data based on provided queries to get the requested data
  • displaying response data in the frontend
