--- layout: post title: "Face Mask Classification" technologies: "Python, NumPy, pandas, PyTorch, opencv" year: "2023" short_description: "Classifying images of people wearing a face mask." project_url: "https://github.com/jimwei6/AI-ML/blob/main/Face_Mask_Classification.ipynb" --- This is a project for Advanced Data Science at The University of Waseda during my exchange. The project compares machine learning models using SIFT features with models using only pixel values on the task of face mask classification.

SIFT Models:
  • Logistic Regression
  • Multi-Layer Perceptron
Pixel Models:
  • Flattened Multi-Layer Perceptron
  • Convolutional Neural Network (LENET)

Dataset: https://www.kaggle.com/datasets/ashishjangra27/face-mask-12k-images-dataset
